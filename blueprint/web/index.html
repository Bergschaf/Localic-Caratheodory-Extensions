Lemma 1 Frame
#

A Frame can be viewed as a Category. There exists a morphism between \(A\) and \(B\) iff \(A \le B\).

Definition 2 Top -> Frame
#

Uses

  • Lemma 1

There exists a contravariant Functor from a topological Space to the corresponding Frame Category with open Sets as objects. \(f: X \rightarrow O(X)\)

Definition 3 \(f^*\) and \(f_*\)
#

Uses

  • Definition 2

For every continuous Function \(f : X \rightarrow Y\) between topological Spaces, there exists a pair of functors \((f^*,f_*)\).

\begin{gather*} f* = f^{-1} : O(Y) \rightarrow O(X)\\ f_* : O(X) \rightarrow O(Y) \coloneqq A \mapsto \bigcup _{f^*(v) \le A} v\\ \end{gather*}
Lemma 4 \(f^*\) commutes

Uses

Lemma 7 Embedding
#

Uses

  • Definition 3

(Leroy Lemme 1) \(f: X \rightarrow Y\) is an embedding, if at least one of the following arguemnts holds:

  1. \(f^*\) is surjective

  2. \(f_*\) is injective

  3. \(f^{*}f_* = 1_{O(X)})\)

Lemma 8 Nucleus
#

Uses

  • Definition 3

(Leroy Lemme 3) Let \(e : O(E) \rightarrow O(E)\) be a continuous map. The following arguemnts are equivalent:

  1. There is a space \(X\) and a morphism \(f: X \rightarrow E\) such that \(e = f_*f^*\). (The same holds for embeddings ??)

  2. \(e\) is idempotent and for every \(U, V \in O(E)\), \(e(U \cap V) = e(U) \cap e(V)\) and \(U \subset e(U)\)

We call \(e\) a nucleus. (Leroy calls it projector Ch 3)

Lemma 9 Image
#

Uses

  • Lemma 8

(Leroy CH 3) A subspace \(Y \subset X\) is defined by a nucleus \(e_Y: O(X) \rightarrow O(X)\), such that \(O(Y) = Im(e_Y) = {U \in O(X) | e_Y(U) = U}\). The corresponding embedding (???) is \(i_X : O(Y) \rightarrow O(X)\). \(i^*_X(V) = e_X(V)\), \((i_X)_*(U) = U\)

Lemma 10 factorisation
#

(Leroy Lemme 2) Let \(i : X \rightarrow E\) be an embedding and \(f: Y \rightarrow E\) be a morphism of spaces. To have \(f\) factor through \(i\), it is necessary and sufficient that \(i_*i^*(V) \le f_*f^*(V) for all V \in O(E)\).
