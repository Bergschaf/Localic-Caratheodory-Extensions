Lemma 7 Embedding
#

Uses

  • f_surjective_injective

(Leroy Lemme 1) \(f: X \rightarrow Y\) is an embedding, if at least one of the following arguemnts holds:

  1. \(f^*\) is surjective

  2. \(f_*\) is injective

  3. \(f^{*}f_* = 1_{O(X)})\)

Definition 8 Nucleus
✓
#

Uses

  • Definition 1

Lean declarations

  • Nucleus

A nucleus is a map \(e : O(E) \rightarrow O(E)\) with the following three properties:

  1. \(e\) is idempotent

  2. \(U \le e U\)

  3. \(e(U \cap V) = e(U) \cap e(V)\)
